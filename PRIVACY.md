# Security & Privacy Policy

## Data Protection

### What We Collect
- Email address (for authentication only)
- IP address (for security and abuse prevention)
- Content you submit (stories, opinions, predictions)
- Submission metadata (timestamps, district information)

### What We Don't Collect
- Personal identifying information (names, phone numbers, addresses)
- Browsing history outside The City
- Location data
- Device identifiers
- Demographic information

### Data Storage
- All data encrypted at rest with industry-standard encryption
- Data stored in Supabase PostgreSQL database
- Automatic daily backups maintained
- Point-in-time recovery available

## Anonymity Guarantees

### How We Protect Your Identity
1. **No Profile Information**: We don't store or display any personal details
2. **Pseudonymous Operations**: All contributions linked only to system-generated user IDs
3. **No IP Logging**: IP addresses used for security, not stored with content
4. **Server-Side Anonymization**: All identifying information stripped before storage

### What We Cannot Guarantee
- If you include identifying information in your content, it will be visible to others
- Law enforcement with valid legal process can compel data disclosure
- Metadata (timestamps) may be used for content moderation

## Content Moderation

### Automated Systems
- AI models scan for harmful content without storing personal data
- Flagged content sent to human reviewers only
- Flagging decisions not linked to your identity

### Manual Review
- Admin team reviews flagged and reported content
- Reviews conducted confidentially
- Admin logs maintained but not shared

### User Reports
- Other users can report content anonymously
- Reports don't expose reporter identity to content creator
- Report history kept for moderation purposes

## Third-Party Services

### Supabase
- Handles authentication and database
- Complies with GDPR and international standards
- See [Supabase Privacy Policy](https://supabase.com/privacy)

### OpenAI
- AI analysis performed on OpenAI servers
- OpenAI's privacy policy governs API usage
- Content sent to OpenAI is not used for training
- See [OpenAI Privacy Policy](https://openai.com/privacy)

### Vercel
- Application hosting and deployment
- See [Vercel Privacy Policy](https://vercel.com/legal/privacy-policy)

## User Rights

### Your Data Access
- Request your data anytime: hello@cityofstrangers.com
- We'll provide all associated records within 30 days

### Data Deletion
- Delete your account anytime in settings
- Associated personal data removed immediately
- Content may remain (anonymized)
- Can request complete content deletion

### Data Portability
- Export your data in standard formats
- Request assistance with data transfer

## Security Measures

### Technical Safeguards
- HTTPS/TLS encryption for all connections
- bcrypt password hashing
- JWT tokens for session management
- Regular security audits
- Dependency scanning for vulnerabilities

### Access Control
- Least privilege principle for staff access
- Multi-factor authentication for admin accounts
- Audit logging for all data access
- Regular access reviews

### Incident Response
- Incidents reported within 72 hours
- Affected users notified immediately
- Public incident report available

## Policy Updates

We may update this policy periodically. Continued use of The City of Strangers constitutes acceptance of updates.

For questions about privacy and security, contact: privacy@cityofstrangers.com
